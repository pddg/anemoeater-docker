# Anemoeater Docker

Please see https://github.com/yoku0825/anemoeater/tree/master to use anemoeater

## Environment

- Docker
- Docker Compose

## Usage

1. Place your slow query log files into `logs` directory (It's extension must be `.log`).
2. `make up`
3. Open http://127.0.0.1:8000/anemometer/
4. `make stop`
